Stage chez Pierres Feuilles Ciseaux

Ce stage a été réalisé a la fin de ma 1er année de BTS SIO du 18/05/2025 au 27/06/2025. Pour accomplir la mission qui m'a été confié j'ai du concevoir un site web e-commerce pierres-feuilles-ciseaux.fr.

Contexte

L’entreprise Pierres Feuilles Ciseaux, située au 18 rue du Grand Montréjeau à ST Barthélémy d’Anjou 49124, a pour but de vendre des objets artisanaux tel que des bijoux (boucle d'oreille, bague, …) et des objets de décorations comme des veilleuses. Toutes les créations sont faites à base de papier ou de matières recyclées.
Elle est sous la responsabilité d'Aurélie Pivette.recyclées.
L'entreprise est composée d'une seule employée qui est également ma tutrice

Outils utilisée--

Pour la création du site avec les connaissance acquise en 1er année de BTS SIO options SLAM, j'ai du utiliser principalement le CMS WordPress et le plugin WooCommerce.

Nom Description
XAMPP Héberge localement un site WordPress avec serveur Apache et base de données MySQL.
PHP/JAVASCRIPT Ajout de code (shortcode/theme)
MySQL Base de données utilisé pour le site web.
WordPress CMS utilisé pour développer le site avec possibilité d’intégrer des plugins, des thèmes et du code personnalisé.
WooCommerce Permet d’accepter les paiements (CB, Apple Pay, etc.) et de gérer les transactions depuis WordPress.
Stripe Plateforme de paiement externe utilisée pour créer des liens de paiement personnalisés.
Product Filter by WBW Affiche un mini-panier latéral personnalisable sans avoir à utiliser Elementor Pro.
ShopLentor Plugin qui permet de personnaliser les pages WooCommerce (boutique, produit seul, panier, etc.) avec Elementor et ajoute des éléments sur Elementor.
Elementor Page builder visuel utilisé pour construire les pages avec des éléments drag-and- drop personnalisables.

Nom Description
WP Mail SMTP Redirige les emails envoyés par WordPress via un serveur SMTP choisi (comme MailTrap).
MailTrap Outil de test d’envoi d’email en environnement local ou dev, sans réellement envoyer les messages à un utilisateur final.
Yoast SEO Fournit une analyse SEO des pages, pour améliorer leur visibilité sur les moteurs de recherche.
If menu Permet de mettre des conditions au menus créer.
All-in-One WP Migration Outil de sauvegarde et migration de site WordPress en local ou production.
UpdraftPlus Permet de créer des sauvegardes régulières (automatisées ou manuelles), restaurables.
Complianz Gère le consentement RGPD (cookies, conditions légales), génère automatiquement les politiques et la bannière de cookies.
